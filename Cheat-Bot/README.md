# CS-111 Final Chat Bot
CS 111 final is online :)

## Setup

1. Clone repo
```shell
git clone https://github.com/Marvin-Deng/CS111.git
```

2. Enter directory
```shell
cd Cheat-Bot
```

4. Create virtual environment
```shell
python -m venv venv
```

5. Install requirements
```shell
pip install -r requirements.txt
```

## Running

```shell
streamlit run app.py
```
